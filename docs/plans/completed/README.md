# Completed Plans

This folder archives execution plans that are done, superseded, or otherwise no longer active.

Rules:

- keep the original plan content intact enough to preserve implementation history
- add a closeout note before moving a plan here
- resolve or explicitly answer any previously open questions
- do not move a plan here if real implementation or operational work is still unresolved

Use `docs/plans/active/` only for currently in-flight work.
